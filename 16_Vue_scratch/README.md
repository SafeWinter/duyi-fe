# 《Vue 零基础到实战开发宝典》简介



## 1 基本情况

本套课程为渡一在 B 站发布的公开课，官网也有免费入口。

笔记主要用于巩固上一篇知识点，查漏补缺。

为精简仓库体积，课程内所有练手项目均采用 `git branch + tag` 的形式迭代更新；其中 `tag` 与课时同步，`branch` 分支则主要用于 `DIY` 拓展练习。
