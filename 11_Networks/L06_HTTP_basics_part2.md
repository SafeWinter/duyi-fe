## 1 状态码的分类

| 分类  | 分类描述                                       |
| :---- | :--------------------------------------------- |
| `1**` | 信息，服务器收到请求，需要请求者继续执行操作   |
| `2**` | 成功，操作被成功接收并处理                     |
| `3**` | 重定向，需要进一步的操作以完成请求             |
| `4**` | 客户端错误，请求包含语法错误或无法完成请求     |
| `5**` | 服务器错误，服务器在处理请求的过程中发生了错误 |

实际工作中，大多数接口的响应码都是 `200 OK`，具体请求成功与否，需在响应 `Body` 中的另行查看。
