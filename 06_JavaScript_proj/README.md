对于每个案例，分为四步完成：

1. 看案例效果演示，思考如何实现，思考和尝试时间不低于 30min；
2. 看老师讲解，随时暂停视频跟写，确保理解每句代码的作用，直到完成所有代码；
3. 关闭视频，看着跟写的代码，复述整个实现逻辑，越详细越好，复述过程录音；
4. 新开工程，根据之前的理解和录音，自行完成整个效果的代码；

每个案例实现后放入各自的 `demo` 文件夹，笔记只对关键逻辑进行归纳总结。
