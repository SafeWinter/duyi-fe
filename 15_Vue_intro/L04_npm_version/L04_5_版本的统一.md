# L04-05：（必看）保持一致的版本



> 本节为后期袁老师的增补内容，主要解决后续章节练手项目的工具版本问题。



`Node.js` 版本：`v12.22.12`（实测有问题，宜改为 `v14.21.3`）

`@vue/cli` 版本：全局 `v4.5.8`：

```shell
# 验证版本
vue --version  # 应输出：@vue/cli 4.5.8
# 卸载其他版本
npm uninstall -g @vue/cli
# 下载指定版本
npm i -g @vue/cli@4.5.8
```



做练手项目时，最好直接复制课件中的 `package.json` 配置。然后清空 `node_modules` 文件夹，重新安装项目依赖。
